# FItVerse_Web
작업 후 항상 깃허브에 업데이트 해주세용

-----------
### 0. 깃허브 자료 다운로드
- cmd 에서 본인 pc에서 FItVerse_Web 파일을 설치할 공간으로 이동 후
```
git clone https://github.com/DevChoco/FItVerse_Web.git
```
- vsc에서 설치한 폴더 열기
### 1. 설치
- nodejs 설치 - https://nodejs.org/ko

### 2. 라이브러리 설치
- vsc 터미널에서
```
npm install
```

### 3. node js 실행
- vsc 터미널에서
```
nodemon index.js
```
